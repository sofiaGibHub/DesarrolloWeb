Etiquetas html

Existen otros tipos de etiquetas para crear una pagina web en HTML 5 que nos permiten encuadrar otros elementos dentro de las paginas web. Algunas de las mas usadas son :
  1. <a>

    2. Permite anclar una direccion URL
    Ejemplo:

    SOY UNA LIGA

  2. <abbr>

    3. Representa una abreviacion o acronimo el atributo obcional title puede ampliar o describir la abreviatuara. si esta presente el atributo title tiene que contener la descripcion completa y nada mas Ejemplo:

    Yo hago HTML

  3. <addres>

    4. Define la informacion del contacto del autor/propietario de un documento u articulo Ejemplo:

    Escrito por Sofia

  4. <article>

    5. Un articulo debe de tener sentido por si mismo y debe de ser posible distribuirlo independientemente del resto del sitio Fuentes potenciales para elemento article :

    • Publicacion en el foro
    • Entrada en el blog
    • Noticia

    Nota: <article> no se muestra como algo especial en un navegador. Sin embargo. puede usar CSS para darle disenio
    Ejemplo article con css

  5. <aside>

    6. Define algun contenido ademas del contenido que se coloca. El contenido aparte debe

    7. Agrega control de audio como reproduccion, pausa y volumen.El source permite especifiocar archivos de audio alternativos entre los que puede elegir el navegador.
    Ejemplo:     TECLADO

  7. <button>
