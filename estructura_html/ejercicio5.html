Etiquetas html

Existen otros tipos de etiquetas para crear una pagina web en HTML 5 que nos permiten encuadrar otros elementos dentro de las paginas web. Algunas de las mas usadas son :
  1. <a>

    2. Permite anclar una direccion URL
    Ejemplo:

    SOY UNA LIGA

  2. <abbr>

    3. Representa una abreviacion o acronimo el atributo obcional title puede ampliar o describir la abreviatuara. si esta presente el atributo title tiene que contener la descripcion completa y nada mas Ejemplo:

    Yo hago HTML

  3. <addres>

    4. Define la informacion del contacto del autor/propietario de un documento u articulo Ejemplo:

    Escrito por Sofia

  4. <article>

    5. Un articulo debe de tener sentido por si mismo y debe de ser posible distribuirlo independientemente del resto del sitio Fuentes potenciales para elemento article :

    • Publicacion en el foro
    • Entrada en el blog
    • Noticia

    Nota: <article> no se muestra como algo especial en un navegador. Sin embargo. puede usar CSS para darle disenio
    Ejemplo article con css

  5. <aside>

    6. Define algun contenido ademas del contenido que se coloca. El contenido aparte debe
    estar indirectamente relacionado con el contenido circulante diseñar el aside


    Ejemplo aside con css: aside
  6. <audio>

    7. Agrega control de audio como reproduccion, pausa y volumen.El source permite especifiocar archivos de audio alternativos entre los que puede elegir el navegador.
    Ejemplo:

  7. <button>
    8. Define un boton en el que puede hacer click Dentro de un <button> puedes poner texto (y etiquetas como <i> , <b> <strong>, <img>, etc)Esto no es posible con un boton creado con <input> Sugerencia: Especificar siempre el type de un button, para indicar a los navegadorea que tipo de boton es.

    Consejo: Puede diseñar botones facilmente con CSS



  8. <form>

    9. Es un contenedor para diferentes tipos de elementos de entrada, como campos de texto, casillas de verificacion, botones de opcion, botones de envio, etc

    • <input type ="text" >
      • Muestra el campo de entrada de texto de una sola linea

    • < input type radio >
      • Muestra un boton de radio para seleccionar de muchas opciones
      Escolaridad
    • <input type=submmit >
      • Muestra una casilla de verificacion, para seleccionar una o varias opciones
    • <input type button >
      • Muestra un boton para hacerle click
      Oprime
  9. <footer>

    10. Representa un contenedor de contenido introductorio o un conjunto de enlaces de navegacion normalmente contiene :
  10. Informacion de auditoria
  11. Informacion de registro
  12. Informacion del contacto
  13. Mapa del sitio
  14. Volver a enlaces superiores
  15. Documentos relacionados
